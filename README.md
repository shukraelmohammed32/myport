# Elite Engineer Portfolio (Next.js)

High-quality developer portfolio built with Next.js App Router, Tailwind CSS, and Framer Motion.

## Stack

- Next.js (App Router)
- TypeScript
- Tailwind CSS
- Framer Motion
- Lucide React icons

## Run locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Structure

- `app/`: route pages and layout
- `components/`: reusable UI/layout/section components
- `data/`: typed local portfolio content
- `public/`: static assets
- `styles/`: global style tokens and utility classes
